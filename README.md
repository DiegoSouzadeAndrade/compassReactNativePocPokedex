# Pokedex

 

## To run this project 
 

0 - Setup git
 

1 - clone this repository
 

2 - yarn
 

3 - npx react-native run-android
 

or


4 - npx react-native start --port 8084 --reset-cache

5 - npx react-native run-android --port 8084

## For build

1 - npx react-native run-android --verbose --variant=release
